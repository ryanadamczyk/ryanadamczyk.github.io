# Tech Stack - Personal Website

## Core Technologies
- **Static Site Generator:** [Jekyll](https://jekyllrb.com/) - Used for building the static site from Markdown and Liquid templates.
- **Programming Language:** [Ruby](https://www.ruby-lang.org/) - The underlying language for Jekyll and its ecosystem.
- **Package Management:** [RubyGems](https://rubygems.org/) - Managed via `Gemfile` and `tale.gemspec`.

## Frontend & Styling
- **CSS Preprocessor:** [SCSS (Sass)](https://sass-lang.com/) - Utilizing a modular architecture for theme customization.
- **Templating Engine:** [Liquid](https://shopify.github.io/liquid/) - Standard Jekyll templating for layouts and includes.
- **Theme:** [Tale](https://github.com/chesterhow/tale) - A minimalist Jekyll theme focused on typography.

## Build & Deployment
- **CI/CD:** [GitHub Actions](https://github.com/features/actions) - Automated deployment via `pages-deploy.yml`.
- **Hosting:** [GitHub Pages](https://pages.github.com/) - Hosting provider for the static site.

## Development Tools
- **Testing:** [HTML-Proofer](https://github.com/gjtorikian/html-proofer) - Used for validating HTML and checking links.
