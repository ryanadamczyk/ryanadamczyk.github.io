# Product Definition - Personal Website

## Initial Concept
This repo contains the code for my personal website, [radamczyk.com](https://radamczyk.com). This website was built using the Jekyll theme as base with lots of custom SCSS [Tale](https://github.com/chesterhow/tale).

## Target Audience
- **Potential Employers and Recruiters:** Showcasing professional experience and technical skills.
- **Technical Community:** Sharing in-depth tutorials, homelab configurations, and security best practices.
- **Personal Acquaintances:** Providing a central hub for personal updates and contact information.

## Core Features
- **Technical Tutorials & Guides:** Comprehensive documentation of Linux hardening (NIST 800-171), Homelab setups, and automation (Power Automate).
- **Professional Portfolio:** A digital resume and project showcase.
- **Personal Blog:** A space for long-form thoughts and periodic updates.
- **Advanced Search:** (Future) Enhanced search capabilities to help users find specific technical content across a growing library of posts.

## Design Goals
- **Minimalist Aesthetic:** Maintaining a clean, content-focused layout based on the Tale theme.
- **Readability:** Prioritizing clear typography and a distraction-free reading experience for technical tutorials.

## Success Criteria
- Documentation of technical processes for future personal reference.
- Clear demonstration of expertise in IT, Security, and Systems Administration.
- Providing high-value, accessible guides for the homelab and IT community.
